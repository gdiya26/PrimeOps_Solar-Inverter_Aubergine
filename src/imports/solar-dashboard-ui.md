Design a **modern, dynamic AI-powered dashboard UI** for a **Solar Plant Inverter Failure Prediction Platform**. This is a real-time monitoring and predictive maintenance system powered by machine learning and generative AI.

The UI must be **interactive, animated, and dynamic**, not a static dashboard mockup.

General Design Style:

* Dark futuristic theme suitable for energy / AI platforms
* Clean, modern layout similar to industrial monitoring systems
* Smooth animations, hover states, and micro-interactions
* Use glowing accents and subtle gradients to represent energy flow
* Components should feel like a **live control center**

Color Theme:

* Background: #0E1117 (deep dark)
* Solar yellow accents: #FFC107
* Green (healthy systems): #00E676
* Red (critical alerts): #FF5252
* Electric blue charts: #1E88E5
* Soft glowing gradients for charts and alerts

Typography:

* Modern sans-serif (Inter / Poppins style)
* Clear hierarchy for metrics and alerts

---

MAIN HOMEPAGE (Control Center Dashboard)

The homepage should feel like a **real-time solar operations command center**.

Top section: Live status metrics cards with animated counters.

Cards should display:

* Total Inverters
* Inverters Online
* Active Fault Alerts
* Energy Generated Today
* AI Predicted Failures (next 7–10 days)

Each card should:

* Animate when values update
* Have glowing borders when alerts occur
* Show small trend arrows (↑ ↓)

Add subtle motion such as pulsing indicators for live systems.

---

LIVE SOLAR PLANT OVERVIEW

Add an **interactive solar plant system visualization**.

Show:
Solar Panels → Strings → Inverters → Grid

Each inverter node should:

* Change color based on health
* Show tooltip on hover
* Pulse red when predicted failure risk is high

Clicking an inverter opens a **side panel with detailed diagnostics**.

---

REAL-TIME DATA CHARTS

Add animated charts that update in real time.

Charts to include:

* Power Output Trend

* Inverter Temperature

* Grid Frequency

Charts should:

* Animate when data updates
* Show hover tooltips
* Highlight anomalies

AI FAILURE PREDICTION PANEL

Create a dedicated section showing predictions.

Each inverter should display:

Failure Risk Level
Low / Medium / High

Probability Score

Prediction Window

This panel explains WHY the model predicted failure.

Example output:

Show a visual bar chart of top contributing factors.

* Slide open with animation
